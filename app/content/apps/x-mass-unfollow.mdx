## Features
- Bulk unfollow options: Unfollow all accounts or selectively unfollow those who don't follow you back.
- Demo mode: Preview the unfollow process without making any changes to your account.
- Whitelist functionality: Protect specific accounts from being unfollowed by adding them to a whitelist.
- Session control: Configurable option to stop the unfollow process after `n` minutes.

## Usage
- Install the [extension](https://chromewebstore.google.com/detail/x-twitter-mass-unfollow/bidolfkgmbnlnijabkjafdajjpocfhol)
- Visit [https://x.com/following](https://x.com/following)
- Choose **ALL accounts** or **NOT FOLLOWING** for a more targeted approach
- Click **STOP** to instantly pause and halt any unfollowing activity.

## Frequently asked questions
#### Do I need an account to use this extension?
No account is required to use the extension.

#### Is it free?
The extension is free to use and [open source](https://github.com/luqmanoop/twitter-mass-unfollow).

#### How does it work?
Normally, you have to go through the tedious process of clicking unfollow button for every account you want to unfollow. This extension automates that process.

#### What if I don't want to unfollow all accounts?
You can use the whitelist functionality in the options page to protect specific accounts from being unfollowed.

#### Can I get suspended on X (Twitter) for using this extension?
In order to not violate X follow/unfollow [rules](https://help.x.com/en/using-x/x-follow-limit), the extension will stop the unfollow process after one minute & you can restart it again. This is the recommended settings. See the extension options page for more details.

#### Does it collect any data?
No data is collected. The source code is available [here](https://github.com/luqmanoop/twitter-mass-unfollow) and you can modify it as you want.
